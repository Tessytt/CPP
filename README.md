2.printf语句
  printf("我是")----运行出 我是
  printf("我是/t")---运行出 我是/（空格）
  printf("我是/n")---运行出 我是/（转行）

#include<stdio.h>
int main()
{
 printf("我是\t");
 printf("杨\t");
 printf("甜\t");
 printf("甜\n");
 return 0;
}
运行结果----我是 杨 甜 甜
